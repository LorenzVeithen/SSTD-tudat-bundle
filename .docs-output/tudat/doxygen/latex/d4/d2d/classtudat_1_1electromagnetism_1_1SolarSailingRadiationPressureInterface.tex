\doxysubsection{Detailed Description}
Class in which the properties of a solar sailing radiation pressure acceleration model are stored. 

Class in which the properties of a solar sailing radiation pressure acceleration model are stored and in which the current solar sailing radiation pressure is calculated based on the source power and geometry, as well as on the spacecraft solar sail model.
