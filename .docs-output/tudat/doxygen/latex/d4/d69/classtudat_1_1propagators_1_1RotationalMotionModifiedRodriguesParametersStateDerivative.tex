Class for computing the state derivative for rotational dynamics of N bodies. 

Class for computing the state derivative for rotational dynamics of N bodies, using modified Rodrigues parameters from body-\/fixed to inertial frame (in modified Rodrigues parameters format) and angular velocity-\/vector of body expressed in body-\/fixed frame as the rotational state of a single body
