\subsubsection*{template$<$typename Independent\+Variable\+Type$>$\newline
class tudat\+::statistics\+::\+Continuous\+Probability\+Distribution$<$ Independent\+Variable\+Type $>$}

Base class for a continuous random variable. 

Base class for a continuous random variable, providing interfaces for evaluating probability and cumulative distribution functions (inverse cdf is only available for derived class \mbox{\hyperlink{classtudat_1_1statistics_1_1InvertibleContinuousProbabilityDistribution}{Invertible\+Continuous\+Probability\+Distribution}}). Specific functions are to be implemented by derived class. Derived class may be used for random number generation through \mbox{\hyperlink{classtudat_1_1statistics_1_1ContinuousRandomVariableGenerator}{Continuous\+Random\+Variable\+Generator}} class.
