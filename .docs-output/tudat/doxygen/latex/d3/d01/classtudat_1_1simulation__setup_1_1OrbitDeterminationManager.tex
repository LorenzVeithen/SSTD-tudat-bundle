Top-\/level class for performing orbit determination. 

Top-\/level class for performing orbit determination. All required propagation/estimation settings are provided to this class, which then creates all objects needed for the propagation and estimation process. The parameter estimation itself is performed by providing measurement data and related metadata (as \mbox{\hyperlink{classtudat_1_1simulation__setup_1_1PodInput}{Pod\+Input}}) to the estimate\+Parameters function.
