\doxysubsection{Detailed Description}
Base class for computing the partial derivatives of light-\/time corrections w.\+r.\+t. estimated parameters. 

Base class for computing the partial derivatives of light-\/time corrections w.\+r.\+t. estimated parameters. A single derived class is implemented per light time correction type. The same derived class is used for computing partials of single correction type w.\+r.\+t. any number of physical parameters
