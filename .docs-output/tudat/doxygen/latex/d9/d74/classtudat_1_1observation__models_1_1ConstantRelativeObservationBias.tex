class tudat\+::observation\+\_\+models\+::\+Constant\+Relative\+Observation\+Bias$<$ Observation\+Size $>$}

Class for a constant relative observation bias of a given size. 

Class for a constant relative observation bias of a given size. For unbiases observation h and bias A, the biased observation is computed as h .$\ast$ A, where .$\ast$ is the component-\/wise multiplication.
