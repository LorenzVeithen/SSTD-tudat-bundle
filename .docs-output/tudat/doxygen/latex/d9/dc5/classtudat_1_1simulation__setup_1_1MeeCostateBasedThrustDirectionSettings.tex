\doxysubsection{Detailed Description}
@get\+\_\+docstring(Mee\+Costate\+Based\+Thrust\+Direction\+Settings.\+\_\+\+\_\+docstring\+\_\+\+\_\+) 

Class for defining settings for MEE-\/costate based thrust direction guidance. Model details can be found in Kluever (2010) and Boudestijn (2014). The MEE-\/costates are provided for the five slow elements, as a function of time. Constructors for constant costates, and costates from an interpolator, are also provided.
