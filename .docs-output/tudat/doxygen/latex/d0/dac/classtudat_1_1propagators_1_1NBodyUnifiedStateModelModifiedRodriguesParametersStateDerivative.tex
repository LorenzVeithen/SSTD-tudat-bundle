Class for computing the state derivative of translational motion of N bodies, using a Gauss method with unified state model with modified rodrigues parameters (USM6).

Class for computing the state derivative of translational motion of N bodies, using a Gauss method with unified state model with modified rodrigues parameters (USM6). In this method, the derivative of the USM6 elements are computed from the total Cartesian accelerations, with the USM6 elements of the bodies the states being numerically propagated.
