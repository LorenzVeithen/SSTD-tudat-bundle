class tudat\+::numerical\+\_\+integrators\+::\+Runge\+Kutta\+Variable\+Step\+Size\+Base\+Settings$<$ Independent\+Variable\+Type $>$}

Base class to define settings of variable step RK numerical integrator. 

Base class to define settings of variable step RK numerical integrator. From this class, two classes are derived, to define the relative and absolute error tolerances. These can be defined as a scalar or as a vector.
