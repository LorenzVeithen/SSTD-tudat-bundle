\doxysubsection{Detailed Description}
\mbox{\hyperlink{classtudat_1_1geometric__shapes_1_1Capsule}{Capsule}} class. 

Class that defines a \mbox{\hyperlink{classtudat_1_1geometric__shapes_1_1Capsule}{Capsule}} shape, like that of the Apollo or ARD (Advanced Re-\/entry Demonstrator) capsule. The class derives from the \mbox{\hyperlink{classtudat_1_1geometric__shapes_1_1CompositeSurfaceGeometry}{Composite\+Surface\+Geometry}} class, its constituent surface geometries are a \mbox{\hyperlink{classtudat_1_1geometric__shapes_1_1SphereSegment}{Sphere\+Segment}} object for the nose region, a \mbox{\hyperlink{classtudat_1_1geometric__shapes_1_1ConicalFrustum}{Conical\+Frustum}} object for the afterbody, a \mbox{\hyperlink{classtudat_1_1geometric__shapes_1_1SphereSegment}{Sphere\+Segment}} object for the rear cap, and a \mbox{\hyperlink{classtudat_1_1geometric__shapes_1_1Torus}{Torus}} object for the connection between the nose and torus parts. The set\+Capsule( ) function needs to be called after each of the parameters has been set to initialize the capsule.
