\doxysubsection{Detailed Description}
Function to check whether an observation is possible, based on body avoidance angle, as vied from single link end. 

Function to check whether an observation is possible, based on body avoidance angle, as vied from single link end. A typical example of this is the sun avoidance (or solar separation) angle, on which a limit is typically imposed. Calculations in this class computes the angle between the line-\/of-\/sight vector(s) from a single link end, and the vecotr from this link end to the center of mass of the body that is to be \textquotesingle{}avoided\textquotesingle{}. NOTE\+: This class computes the position of the body that is to be avoided in at the time halfway between transmission and reception time of the signal
