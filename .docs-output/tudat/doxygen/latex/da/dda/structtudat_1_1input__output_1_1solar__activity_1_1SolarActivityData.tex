\doxysubsection{Detailed Description}
Solar activity data class. 

Class containing all variables provided in the \href{http://celestrak.com/SpaceData/sw19571001.txt}{\texttt{ http\+://celestrak.\+com/\+Space\+Data/sw19571001.\+txt}} file, describing \char`\"{}space weather\char`\"{} on each day since October 1957. See reference for details on variables.
