Whether all the data from the Spice kernels should be preloaded before the simulation for the interval start epoch to end epoch (true), or whether the data from Spice should be accessed on request at every step (false).

Whether all the data from the Spice kernels should be preloaded before the simulation for the interval start epoch to end epoch (true), or whether the data from Spice should be accessed on request at every step (false). ~\newline
 Preloading Spice data generally results in faster propagations, unless\+: ~\newline
 
\begin{DoxyItemize}
\item The simulation ends much earlier than the specified maximum simulation end epoch. 
\item The integrator step-\/size is very large (in the order of several hours or days).
