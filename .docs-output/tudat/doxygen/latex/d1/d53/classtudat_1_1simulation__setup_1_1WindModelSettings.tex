\doxysubsection{Detailed Description}
Class for providing settings for wind model. 

Class for providing settings for automatic wind model creation. This class is a functional (base) class for settings of wind models that require no information in addition to their type. Wind model classes defining requiring additional information must be created using an object derived from this class.
