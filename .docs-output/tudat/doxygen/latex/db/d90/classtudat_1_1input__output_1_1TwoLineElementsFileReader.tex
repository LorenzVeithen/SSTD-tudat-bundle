\doxysubsection{Detailed Description}
Two-\/\+Line Elements (TLE) file reader class. 

Definition of TLE file reader class. Reads and parses a TLE data file (in the same formatting as files from the Celes\+Trak website) into a map with the name of the satellite as the key and the concatenated two lines with the elements as the value.
