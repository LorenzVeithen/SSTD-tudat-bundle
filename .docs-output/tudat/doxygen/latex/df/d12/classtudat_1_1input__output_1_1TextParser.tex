\doxysubsection{Detailed Description}
A \mbox{\hyperlink{classtudat_1_1input__output_1_1TextParser}{Text\+Parser}} is a \mbox{\hyperlink{classtudat_1_1input__output_1_1Parser}{Parser}} that specialized in parsing text files. 

A \mbox{\hyperlink{classtudat_1_1input__output_1_1TextParser}{Text\+Parser}} provides a set of functions to streamline parsing to simplify the parsing process. The inheriting parsers can chose there prefered way of processing by calling \mbox{\hyperlink{classtudat_1_1input__output_1_1TextParser_a68d2338a4ec8741beecae9ea4559ba49}{Text\+Parser(bool)}}.

NOTE\+: This \mbox{\hyperlink{classtudat_1_1input__output_1_1TextParser}{Text\+Parser}} works with the Field\+Value/\+Field\+Type architecture. For simpler file reading, use, for instance, matrix\+Text\+File\+Reader.
