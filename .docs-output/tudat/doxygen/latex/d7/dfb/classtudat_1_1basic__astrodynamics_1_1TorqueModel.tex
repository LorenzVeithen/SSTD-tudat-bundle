\doxysubsection{Detailed Description}
Pure virtual base class for a model that computes the torque acting on a body. 

Pure virtual base class for a model that computes the torque acting on a body. These models are used in the propagation of the rotational equations of motion. Every specific torque model requires its own derived class.

Update member variables used by the torque model. 

Updates member variables used by the torque model. In the case of torque models containing varying parameters, function-\/pointers returning such a parameter (for instance the Cartesian state of a body) will be set as a member variable. This function evaluates such function-\/pointers and updates member variables to the \textquotesingle{}current\textquotesingle{} values of these parameters. Only these current values, not the function-\/pointers are then used by the get\+Torque function. This pure virtual function must be overridden by derived classes.
