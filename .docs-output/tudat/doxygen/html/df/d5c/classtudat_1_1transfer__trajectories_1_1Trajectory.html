Detailed Description

Base class for computation of trajectories.

This class can compute entire space trajectories of various kinds by concatenating different mission legs. It can compute the total deltaV, but also contains functions that can be used for plotting the trajectory and that of the bodies involved in the trajectory.

void tudat::transfer_trajectories::Trajectory::updateEphemeris ( )

Update the ephemeris.

Sets all the positions and the velocities of the trajectory class and the underlying mission leg classes to new values, that are obtained by extracting the ephemeris at the times that are now in the trajectory variable vector. This is required for re-using the class, without re-initializing it.
