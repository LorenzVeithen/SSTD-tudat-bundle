Detailed Description

Class to compute the acceleration due to tidal dissipation caused by/on this satellite on/by host planet.

Class to compute the acceleration due to tidal dissipation caused by/on this satellite on/by host planet. Depending on the constructor input, an object uses either the tide raised on the satellite, or the tide raised on the planet. The computation is according to Lainey et al. (2007, 2009, 2012, etc.) and uses a single (degree 2) real Love number and a single time lag to include the dissipation in the host planet. The derivation of the model is given by Mignard (1981). For the tide raised on the satellite, the term that includes the angular velocity vector of the deformed body is omitted. Instead, the radial dissipation term is derived by Murray & Dermott (1999, p. 172) to be 3/4 of the tangential dissipation term, see also Caudal et al. (2017).
