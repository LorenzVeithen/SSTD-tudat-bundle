Detailed Description

Class to define settings for observation viability calculator creation.

Class to define settings for observation viability calculator creation. Settinsg are defined by type of viability check, link end at which the check is to be performed, as well as double and string specifiers, which provide any needed specific parameters of the vaibility check (meaning of both parameters is given in constructor doxygen, as well as in-code comments of ObservationViabilityType
