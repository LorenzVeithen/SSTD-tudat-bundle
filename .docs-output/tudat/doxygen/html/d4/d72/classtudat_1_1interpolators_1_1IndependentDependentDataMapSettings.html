Detailed Description

template<typename IndependentType, typename DependentType>
class tudat::interpolators::IndependentDependentDataMapSettings< IndependentType, DependentType >

Class defening the settings to be used to create a map of data (used for interpolation).

Consturctor. The data map will be created by combining values of independent and dependent variables.
