Detailed Description

template<typename StateScalarType, typename TimeType>
class tudat::ephemerides::TabulatedRotationalEphemeris< StateScalarType, TimeType >

Class that computes the current rotational state from tabulated values of the rotational state with the use of an interpolator.

Class that computes the current rotational state from tabulated values of the rotational state with the use of an interpolator The interpolated data consists of the four entries (w,x,y,z) of the quaternion from the target frame to the base frame, and body's angular velocity vector, expressed in its body-fixed frame (target frame). The quaternion is normalized to 1 before being set as the current rotation. Other properties of the rotation are derived from these two quantities.
