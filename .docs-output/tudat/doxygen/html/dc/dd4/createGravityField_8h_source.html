416 // Function to read a spherical harmonic gravity field file
417 /*
418  * Function to read a spherical harmonic gravity field file, returns (by reference) cosine and sine
419  * spherical harmomic coefficients.
420  * The file structure should be as follows: The first line may be a file header with metadata. If this is the case,
421  * the gravitationalParameterIndex and referenceRadiusIndex should be used as input, to communicate which entries in
422  * the list of metadata represents these quantities. If both these variables are NaN, the file is assumed to have no
423  * header. The following lines of the file must have the following structure:
424  * Degree, Order, Cosine Coefficient, Sine Coefficients
425  * Subsequent columns may be present in the file, but are ignored when parsing.
426  * All coefficients not defined in the file are set to zero (except C(0,0) which is always 1.0)
427  * \param fileName Name of PDS gravity field file to be loaded.
428  * \param maximumDegree Maximum degree of gravity field to be loaded.
429  * \param maximumOrder Maximum order of gravity field to be loaded.
430  * \param gravitationalParameterIndex
431  * \param referenceRadiusIndex
432  * \param coefficients Spherical harmonics coefficients (first is cosine, second is sine).
433  * \return Pair of gravitational parameter and reference radius, values are non-NaN if
434  * gravitationalParameterIndex and referenceRadiusIndex are >=0.
435  */
436 std::pair< double, double > readGravityFieldFile(
437  const std::string& fileName, const int maximumDegree, const int maximumOrder,
438  std::pair< Eigen::MatrixXd, Eigen::MatrixXd >& coefficients,
439  const int gravitationalParameterIndex = -1, const int referenceRadiusIndex = -1 );
