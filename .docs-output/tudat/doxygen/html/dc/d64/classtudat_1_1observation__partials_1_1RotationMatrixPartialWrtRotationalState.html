Detailed Description

Class to calculate a rotation matrix from a body-fixed to inertial frame w.r.t. the rotational state.

Class to calculate a rotation matrix from a body-fixed to inertial frame w.r.t. the rotational state, consisting of the associated quaternion elements and body-fixed angular velocity vector.
