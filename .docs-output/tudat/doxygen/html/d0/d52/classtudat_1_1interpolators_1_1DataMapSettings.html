Detailed Description

template<typename IndependentType, typename DependentType>
class tudat::interpolators::DataMapSettings< IndependentType, DependentType >

Class defening the settings to be used to create a map of data (used for interpolation).

Empty constructor. The class can be used to provided a map directly, or as a base class for loading the map using different procedures.
