Detailed Description

@get_docstring(CustomThrustOrientationSettings.__docstring__)

Class for defining custom orientation of thrust (i.e. predefined body-fixed-to-propagation rotation as function of time). Thrust is then computed from body-fixed direction of thrust (defined in ThrustMagnitudeSettings).
