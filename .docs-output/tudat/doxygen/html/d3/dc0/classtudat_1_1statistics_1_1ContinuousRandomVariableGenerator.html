Detailed Description

Random number generator generated directly from inverse cdf function of probability distribution.

This class uses a probability distributio (or more specifically the inverse cumulative distribution function) from which numbers are randomly generated. The inverse cdf function is evaluated from an object of type InvertibleContinuousProbabilityDistribution< double >. The uniform random number generator that is mapped to the random number from the specific distribution is generated from the boost::random::mt19937 generator; its seed may be either pseudo-random (time( 0 )) or user-defined.
