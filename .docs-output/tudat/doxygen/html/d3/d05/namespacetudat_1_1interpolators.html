Compute linear interpolation free function.

Compute linear interpolation.

Computes linear interpolation of data provided in the form of a vector of sorted indepedent variables and an associated vector of dependent variables. The linear interpolation equation used is:

\[ y_{target} = x_{1} * ( 1 - mu ) + x_{2} * mu \]

where \( mu = \frac{ x_{target} - x_{1} } { x_{2} + x_{1} } \) and \( x_{2} > x_{1} \).

Parameters
sortedIndependentVariablesVector of independent variables sorted. in ascending/descending order.
associatedDependentVariablesVector of dependent variables associated with sorted vector of independent variables.
targetIndependentVariableValueTarget independent variable value in vector of sorted independent variables.
Returns
Value of dependent variable associated with target independent value in vector of sorted independent variables.

Solve efficiently tri-diagonal matrix equation.

This functions efficiently solves the matrix equation Ax = b for b, where A is tri-diagonal. The required input are the diagonal and sub/super diagonals of A, as well as the vector b. Input is given as STL vectors. The input diagonals must be at least the same size as right-hand-side. Any additional entries (at end) will be ignored in the algorithm.
