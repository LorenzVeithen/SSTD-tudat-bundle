Detailed Description

Class used for computing an acceleration due to a momentum wheel desaturation.

Class used for computing an acceleration due to a momentum wheel desaturation. The deltaV values for each of the desaturation maneuvers are provided by the user.

Update member variables used by the momentum wheel desaturation acceleration model.

Update member variables used by the momentum wheel desaturation acceleration model. Function pointers to retrieve the current values of quantities from which the acceleration is to be calculated are set by constructor. This function calls and combines their output to compute the acceleration.
