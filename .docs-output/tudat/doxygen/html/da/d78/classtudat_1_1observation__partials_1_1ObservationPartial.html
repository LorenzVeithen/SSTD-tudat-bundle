template<int ObservationSize>
class tudat::observation_partials::ObservationPartial< ObservationSize >

Base class of partial derivative of an observable w.r.t. an estimated parameter.

Base class of partial derivative of an observable w.r.t. an estimated parameter. Calculated partial function must be defined in each derived class for calculation of specific derivative.

Pure virtual function to calculate the observation partial(s) at required time(s) and state(s)

Pure virtual function to calculate the observation partial(s) at required time(s) and state(s). States and times are typically obtained from evaluation of associated observation model. Derived class functions implement this for a specific observable.

Parameters
statesLink end states. Index maps to link end for a given ObsevableType through getLinkEndIndex function.
timesLink end times.
linkEndOfFixedTimeLink end that is kept fixed when computing the observable.
currentObservationValue of the observation for which the partial is to be computed (default NaN for compatibility purposes).
Returns
Vector of pairs containing partial values and associated times.

Implemented in tudat::observation_partials::ObservationPartialWrtArcWiseRelativeBias< ObservationSize >, tudat::observation_partials::ObservationPartialWrtArcWiseAbsoluteBias< ObservationSize >, tudat::observation_partials::ObservationPartialWrtConstantAbsoluteBias< ObservationSize >, and tudat::observation_partials::ObservationPartialWrtConstantRelativeBias< ObservationSize >.
