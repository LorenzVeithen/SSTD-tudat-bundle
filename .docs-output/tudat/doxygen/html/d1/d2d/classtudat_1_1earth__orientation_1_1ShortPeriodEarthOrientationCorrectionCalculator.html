Constructor, requires files defining the amplitudes and fundamental argument multipliers of the variations.

Constructor, requires files defining the sine and cosine amplitudes and fundamental argument multipliers numbers of the variations for both ocean tide and libration variations. A cut-off amplitude (taken as the RSS of the amplitudes) below which amplitudes in the files are ignored can be provided.

Parameters
conversionFactorConversion factor to be used for amplitudes, used to multiply input values, typically for unit conversion purposes.
minimumAmplitudeMinimum amplitude that is read from files and considered in calculations.
amplitudesFilesList of files with amplitudes for corrections
argumentMultipliersFileFundamental argument multiplier for corrections
argumentFunctionFundamental argument functions associated with multipliers, default Delaunay arguments with GMST
